Сравнительные таблицы графических карт AMD (ATI) Radeon 

    Release Date — дата выпуска
    Core — ядро (графический чип)
    On-board Memory — объём локальной памяти
    Interface — интерфейс (шина) подключения
    Core Clock — частота работы ядра
    Memory — локальная память
    Type — тип
    Clock (eff.) — рабочая (effective, эффективная) частота
    Bus — ширина шины данных
    SDR (Single Data Rate) — память с возможностью передачи одного пакета данных за такт
    DDR (Double Data Rate) — память с возможностью передачи двух пакетов данных за такт
    MBW(Memory BandWidth) — пропускная способность памяти
    Config Core (SPU:TMU(TF):ROP) — конфигурация блоков графического процессора
    SPU (Stream Processor Unit) — универсальный потоковый процессор (шейдерный блок)
    TMU (Texture Mapping Unit) — блок текстурирования
    TF (Texture Filtering Unit) - блок текстурной (билинейной) фильтрации
    TA (Texture Address Unit) - блок текстурной адресации
    ROP (Raster Operation Pipe) — блок растеризации
    PPU (Pixel Processor Unit) — пиксельный процессор
    Fill Rate — скорость заполнения
    Processing Power — теоретическая производительность на вычислениях с плавающей запятой
    FP32 (Single Precision) — одинарная точность вычислений с плавающей запятой
    FP64 (Double Precision) — двойная точность вычислений с плавающей запятой
    TDP (Thermal Design Power) — тепловой пакет
    TC — Turbo Cache
    OEM — (Original Equipment Manufacturer) — производитель комплектного оборудования
    Benchmark — Результаты бенчмарков с сайта http://www.userbenchmark.com/
